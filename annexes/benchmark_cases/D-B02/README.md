# D-B02

Financial domain baseline slice. Two spouses without children navigate a shared credit-card debt and housing cost dispute driven by genuine misunderstanding rather than bad faith. The session tests whether Solomon can surface the missing financial facts before packaging, and whether it correctly holds M2 available when the fact gap is severe enough.

This is the only childless financial-domain case in the corpus and the first case where E5 (decision-quality escalation) is the primary likely category.
