# D-B01

Cooperative separation baseline slice. Two recently separated parents navigate logistics coordination disagreements — pickup timing, activity hand-off, and holiday sequencing — in a low-conflict, cooperative-intent context.

This is the primary calibration anchor for the corpus. It defines what good Solomon performance looks like when the system is operating well in the simplest conditions, with no safety concern, no domination, and a wide settlement zone. Evaluators should use this case to calibrate what M0 looks like before scoring harder escalation cases.
